# animated-login-page

[![中文文档](https://img.shields.io/badge/README-%E4%B8%AD%E6%96%87-blue)](./README.zh-CN.md) [![Live Demo](https://img.shields.io/badge/demo-live-22c55e)](https://ningzimu.github.io/animated-login-page/)

A reusable animated login page built with React and Vite.

This project is a standalone implementation of an animated multi-character login experience, designed to be easy to run, customize, and adapt for other products.

## Live Demo

- https://ningzimu.github.io/animated-login-page/

## Features

- Animated multi-character hero section
- Input-aware character interactions
- Responsive layout for desktop and mobile
- Clean split-screen login page structure
- Reusable React-based project setup
- Easy to customize copy, colors, and form behavior

## Why this project exists

The goal of this project is not just to recreate a nice-looking login screen, but to turn that interaction style into a reusable starting point for real projects.

Compared with a one-off demo component, this repository is intended to be:

- easier to run as a standalone app
- easier to modify and extend
- easier to adapt into your own authentication flow
- easier to reuse across multiple projects

## Tech Stack

- React
- Vite
- CSS
